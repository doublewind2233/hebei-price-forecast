# 河北冀南电力 · 下月电价预测报告工具

基于河北电力市场实际结算数据和国网代理购电价，为售电侧客户提供下月电价预测与费用对比分析。

## 功能特性

- **数据预置**：已预置2026年1-3月市场传导价格（24点/小时）
- **国网电价自动生成**：基于河北南网分时电价政策（冀发改能价〔2025〕1115号）自动生成24小时国网电价曲线
- **费用对比**：同一用电量下，市场结算 vs 国网代理购电的总费用对比
- **图表展示**：双折线图展示本月费用对比与下月预测对比
- **客户用电量输入**：支持Excel粘贴（24点电量数据）

## 数据更新流程

本工具采用**预置数据+自动生成**模式，无需每月手动上传：

### 市场传导价格
- **来源**：河北电力交易中心每月下月初发布上月数据（PDF）
- **更新**：每月提取24点数据，更新 `hebei_price_forecast_v2.html` 中的 `PRESET_MARKET_PRICES` 对象
- **当前预置**：2026年1月、2月、3月（完整）

### 国网代理购电价
- **来源**：国家电网每月月底发布下月代理购电价表
- **更新**：提取平段价格，工具自动根据月份、季节规则生成24小时曲线
- **当前预置**：2026年1-4月平段价格（自动生成算法已内置）

## 使用方法

1. 选择月份（1-4月）
2. 在"客户用电量输入"表格中粘贴24点电量数据（可从Excel直接复制）
3. 点击"生成对比报告"
4. 查看费用对比表格与双折线图

## 技术说明

- 纯前端实现（HTML/CSS/JavaScript）
- 图表库：Chart.js（CDN加载）
- 无后端依赖，所有数据预置在页面中
- 支持本地文件系统运行

## 维护说明

每月更新流程：
1. **月底**：获取国网下月代理购电价表，记录平段价格
2. **月初**：获取交易中心上月市场传导价PDF，提取24点数据
3. **更新工具**：修改 `PRESET_MARKET_PRICES` 和 `PRESET_GRID_PRICES` 中的对应月份数据
